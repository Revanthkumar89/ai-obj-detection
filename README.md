# ai-obj-detection
Developed a browser-based AI application for real-time object detection using device camera input. Implemented interactive controls such as toggle switch for AI activation and frame rate adjustment for performance optimization. Ensured cross-browser compatibility and delivered a lightweight, user-friendly interface for real-time detection.
Developed a real-time AI Object Detection web application using Computer Vision and Machine Learning concepts.
Implemented object detection functionality using TensorFlow and COCO-SSD model.
Integrated OpenCV and JavaScript for live camera-based object recognition.
Designed an interactive web interface with AI toggle controls and frame rate adjustment features.
Enabled real-time detection through browser camera access and optimized performance for smooth execution.
Used HTML, CSS, and JavaScript for frontend development and user interaction.
Managed project version control and deployment using Git and GitHub.
Applied concepts of Deep Learning, Image Processing, and Real-Time Video Analysis.
